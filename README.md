# Crownbound 👑

Crownbound is a 2D action-adventure game developed using Unity. The game focuses on dynamic combat mechanics, smooth character animations, and strategic gameplay involving both melee and ranged attacks.

## 🎮 Features
- ⚔️ Sword-based melee combat system
- 🏹 Bow and arrow ranged attack mechanics
- 🚶 Smooth player movement with synced animations (idle, walking, attacking)
- 🎭 Custom character animations (sprite-based)
- 🧠 Intelligent input handling using Unity Animator and Rigidbody2D
- 📊 Result visualization system inspired by graph-based outputs

## 🛠️ Built With
- Unity (2D Game Engine)
- C#
- Visual Studio Code

## 🚀 Gameplay Overview
The player can switch between close-range and long-range combat styles:
- Use sword attacks for close combat
- Use bow attacks for enemies at a distance

Animations are fully synchronized to enhance realism and responsiveness.

## 📂 Project Structure
- `Scripts/` → Player movement, combat logic, and systems
- `Animations/` → Sprite sheets and animation controllers
- `Assets/` → Game assets including characters and UI
- `Scenes/` → Game levels and environments
